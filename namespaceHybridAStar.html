typedef int(* HybridAStar::DubinsPathSamplingCallback)(double q[3], double t, void *user_data)

Callback function for path sampling

Note
the q parameter is a configuration
the t parameter is the distance along the path
the user_data parameter is forwarded from the caller
return non-zero to denote sampling should be stopped
